로컬 컴퓨터에서 먼저 git 저장소 생성하고 나중에 서버에 등록하기

1. 소스트리에서 'create' 버튼 클릭
2. git 저장소 위치 및 프로젝트명 설정
3. 저장소 생성 후 버전 관리할 파일 생성
4. 커밋을 눌러 버전 관리 시작
5. github 사이트에서 원격 저장소 생성 (로컬 컴퓨터의 저장소 이름과 동일하게 생성)
6. 원격 저장소 url 복사하기
7. 소스트리에서 '설정'버튼 클릭
8. 원격지 주소 추가
9. 원격지 이름을 'origin' 으로 설정 (혹은 디폴드 이름 사용하면 origin 이 자동입력됨)
10. 원격지 url 붙여넣기
11. url 에 '토큰정보' 추가 후 '확인'버튼 클릭
12. 소스트리에 'push' 버튼 클릭해서 정상적으로 동작하는지 확인
